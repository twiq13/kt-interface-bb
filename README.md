KT's UI Pack - WoW Ascension 3.3.5
=======================================

1️⃣ Si vous n'avez pas encore d'addons copiez le dossier Interface et Font dans votre dossier WoW :  
   Exemple : C:\Program Files\Ascension Launcher\resources\client  
   (Le dossier Interface contient les addons et Font vous permet d'avoir ma police d'écriture)  
   
   Si vous avez déjà un fichier Interface :  
   → Faites une sauvegarde avant !
   
2️⃣ Lancez le jeu, connectez-vous, et sélectionnez les profil :  
   - ElvUI : /elvui → Profiles → Import → (copiez depuis le fichier Elvui_Profile.txt) 
   - Details : /details → Profiles → Import → (copiez depuis le fichier Details_Profile.txt)  
   - WeakAuras : /wa → Import → (copiez depuis le fichier All_Weakauras.txt)  

3️⃣ Redémarrez WoW si les addons ne se chargent pas.
